Tipos de datos en programación JS

Bienvenidos a la página sobre tipos de datos en JavaScript. Aquí aprenderás sobre los diferentes tipos de datos que puedes utilizar en tus programas.

1. Números: Representan valores numéricos, tanto enteros como decimales.
2. Cadenas de texto (Strings): Representan secuencias de caracteres, como palabras o frases.
3. Booleanos: Representan valores de verdad, es decir, verdadero (true) o falso (false).
4. Arreglos (Arrays): Estructuras que pueden contener múltiples valores en una sola variable.
5. Objetos: Estructuras que permiten almacenar datos en pares clave-valor.
6. Nulo (Null): Representa la ausencia intencional de cualquier valor.
7. Indefinido (Undefined): Indica que una variable ha sido declarada pero no se le ha asignado un valor.

Ejemplos prácticos

A continuación, se presentan algunos ejemplos prácticos de cada tipo de dato:

  • Números: let edad = 25;
  • Cadenas de texto: let nombre = "Juan";
  • Booleanos: let esEstudiante = true;
  • Arreglos: let frutas = ["manzana", "banana", "naranja"];
  • Objetos: let persona = { nombre: "Juan", edad: 25 };
  • Nulo: let valorNulo = null;
  • Indefinido: let valorIndefinido;
